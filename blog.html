What are HTTP – Hypertext Transfer Protocol?

HTTP are communication between webservers and clients.It was originally designed to fetch html data (in 1990s). Transport Layer Security (TLS) or, formerly, Secure Sockets Layer (SSL) In this the communication should be through http protocol and it can deliver any type of data .It is connection less : after making request , client disconnects from server and Stateless : client – server knows only during a transaction.

    Request Types

  • GET – Requesting resource from server
  • POST – submitting a resource to a server (e.g. file uploads)
  • PUT -As POST but replaces a resource
  • DELETE-Delete a resource from a server
  • OPTIONS -Get the options for the resource
  • HEAD – As GET but only return headers and not content
  • PATCH -Apply modifications to a resource
  • TRACE -Performs message loop-back
